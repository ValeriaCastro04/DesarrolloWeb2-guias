Guías semanales

Guía 1

En la primera guía de la materia de desarrollo web II con memo, se pretende evaluar el uso que el estudiante le da al FrameWork Tailwind CSS para estilizar páginas web.

  • Tarjetas de precios
  • Producto
  • Galería de imágenes
  • Formulario de inicio de sesión
  • Clipboard Landing Page
  • Ejercicio extra

Guía 2

En esta guía se exploró el uso de la herramienta React para renderizar sitios web y cómo se implementa en el desarrollo, a su vez se usa el deploy del sitio.

  • Deploy: Guitar LA

Guía 3

En esta guia se exploró el uso de React como una tool para renderizar sitios web, así como el uso de sus propiedades para heredar props y utilizar apropiadamente los hooks. Por último, se aprovecho el LocalStorage para asegurar la persistencia de los datos.

  • Deploy: Palomitas de papel

Guía 4

Esta guía se centró en el uso de los contextos y reducers como métodos para consolidar la lógica de actualización del estado de varios componentes. Estos nos sirvieron para compartir estados sin necesidad de utilizar hooks y estructurar de mejor manera el código. El objetivo era utilizar Context API para manejar estados globales.

  • Deploy: planificador de gastos
